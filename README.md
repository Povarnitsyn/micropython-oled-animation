# MicroPython OLED Animation Player: OLED SSD1306 ESP32 Blender

Проект по созданию и воспроизведению плавной 3D-анимации на OLED-дисплее SSD1306 (128x64). Включает в себя полный цикл: от процедурного рендеринга в Blender до асинхронного вывода на контроллере.
Протестировано на ESP32-S3-DevKitC-1 N16R8.

## 🛠 Технологический процесс

### 1. 3D-моделирование и Рендер (Blender)
*   **Движок:** Cycles, разрешение 128x64.
*   **Материал:** Нода `Pointiness` через множитель `Math` подсвечивает извилины, создавая глубокий контраст.
*   **Четкость:** Отключено сглаживание (`Pixel Filter Width: 0.01`), что дает «бритвенно-острые» пиксели без мыла.
*   **Освещение:** Мозг светится сам (Emission), что убрало лишние шумы на фоне.

### 2. Подготовка данных (Python + Pillow)
Используется скрипт конвертации PNG-последовательности в бинарный файл `brain.bin`:
*   **Threshold:** Фильтрация паразитных ореолов и чистка «белого» от черных точек шума.
*   **Дизеринг:** Алгоритм `Floyd-Steinberg` для создания иллюзии объема на 1-битном экране.
*   **Формат:** Данные упакованы в `MONO_HLSB`, оптимизированный для драйвера SSD1306.

### 3. Прошивка (MicroPython)
*   **Асинхронность:** Реализован `display_task` на `uasyncio`, который читает файл по кадрам, не забивая RAM.
*   **Оптимизация видео:** Частота шины SPI — 4-8 МГц. 
*   **Video Friendly:** Настроен разгон внутреннего генератора дисплея (регистр `0xD5`), что убирает мерцание при съемке на камеру.

## 📂 Структура проекта
*   `main.py` — основной код для MicroPython.
*   `brain.bin` — готовая анимация (бинарный файл).
*   `process_images/` — папка со скриптом конвертации и исходными кадрами (`frames/`).

## 🚀 Инструкция по запуску

### Схема подключения (SPI)

| OLED Pin | ESP32/RP2040 Pin | Описание |
|----------|------------------|----------|
| D0 (SCLK)| GPIO 8           | Clock    |
| D1 (MOSI)| GPIO 9           | Data In  |
| RES      | GPIO 10          | Reset    |
| DC       | GPIO 11          | Data/Cmd |
| CS       | GPIO 12          | Chip Sel |
| MISO     | GPIO 3           | (Dummy)  |

### Установка
1.  Скачайте драйвер [ssd1306.py](https://github.com/stlehmann/micropython-ssd1306/blob/master/ssd1306.py).
2.  Сохраните его на устройство под именем `display_driver.py`.
3.  Загрузите `main.py` и `brain.bin`.
